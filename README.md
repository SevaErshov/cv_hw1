# Мини-система стабилизации камеры

В качестве ДЗ был выбран вариант А: инструмент для стабилизации видео с использованием оптического потока Farnebäck и сглаживания траектории камеры.

## Installation
```bash
git clone https://github.com/SevaErshov/cv_hw1.git
cd cv_hw1
poetry install
```

## Example of usage
```bash
poetry run python3.12 run.py input=video/my_video.mov output=video/my_stable.mp4 border=50
```

Визуализация траектории и ее сглаживание доступно в папке visualizations.

## Отчет по выполнению ДЗ
Для выполнения ДЗ использовалось черно-белое видео с ручной съемкой. На видео представлена текстурированная стена.

Для вычисления плотного оптического потока использовался алгоритм Farnebäck. Далее при помощи построения аффинной модели (RANSAC)оценивалось глобальное движение. Траектория рассчитывается как кумулятивная сумма трансформаций и ее сглаживание осуществляется при помощи скользящего среднего (radius=15). При помощи warp происходит компенсация движения. После этого этапа возникают черные рамки, поэтому видео обрезается и возвращается к исходному размеру.

### Наблюдения и выводы
После применения алгоритма видео стало значительно более стабильным.
На видео представлена текстурированная стена (оптичекий поток хорошо справляется с текстурированными объектами), поэтому дрожание почти исчезло, но всё же небольшая тряска ещё заметна.

Для удаления черных рамок пришлось обрезать края.
